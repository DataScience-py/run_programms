# Download Guide

## Python Script (.py)

To download the Python script:

1. **Visit the Repository**:
   - Go to the repository where the script is hosted.

2. **Download**:
   - Click on the script file and download it to your local machine.

3. **Run the Script**:
   - Make sure you have Python 3.x and `psutil` installed.
   - Run the script using the following command:

     ```bash
     python script_name.py
     ```

## Executable File (.exe)

To download the executable file:

1. **Visit the Releases Page**:
   - Go to the releases page of the repository or the hosting site.

2. **Download**:
   - Find the `.exe` file in the assets and download it.

3. **Run the Executable**:
   - Simply double-click the downloaded `.exe` file to run it.

4. **Requirements**:
   - Ensure that all required dependencies are available or bundled with the executable.

## Notes

- Ensure that you have the necessary permissions to run the `.exe` file on your system.
- For any issues or errors, refer to the [README.md](README.md) file for troubleshooting.
